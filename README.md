# 🌐 Holberton School – Networking

This repository contains projects from the **Holberton School Networking curriculum**, focused on understanding how computers communicate over networks, internet protocols, and system-level networking concepts.

---

## 📌 Overview

This project builds strong foundations in:

- 🌍 Internet fundamentals
- 📡 Networking protocols (TCP/IP, HTTP, DNS)
- 🔎 Network troubleshooting
- 🧠 OSI & TCP/IP models
- ⚙️ Practical networking commands

---

## 📂 Project Structure

```
holbertonschool-network/
│
├── basics/              # Intro to networking concepts
├── protocols/           # TCP, UDP, HTTP, DNS
├── tools/               # ping, netstat, traceroute, curl
```

---

## 🧠 Key Concepts Learned

### 🌐 Networking Basics
- What is a network
- LAN vs WAN
- IP addressing (IPv4)

### 📡 Protocols
- TCP vs UDP
- HTTP & HTTPS
- DNS resolution

### 🔎 Troubleshooting Tools
- ping
- traceroute
- netstat
- curl

### 🧠 Models
- OSI model (7 layers)
- TCP/IP model
